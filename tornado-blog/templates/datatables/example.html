Bootstrap styling

This example shows DataTables and the Responsive extension being used with the Bootstrap framework providing the styling. The DataTables / Bootstrap integration prove seamless integration for DataTables to be used in a Bootstrap page.

Note that the dt-responsive class is used to indicate to the extension that it should be enabled on this page, as responsive has special meaning in Bootstrap. The responsive option could also be used if required.
